Java_Fundamental_Programs

This repository contains Java programs that I have written while learning the fundamentals and core principles of the Java programming language.
It is intended to track my learning progress and to practice basic to advanced concepts of Java 

Try It Yourself

You can easily clone this repository and run the programs on your local machine.

Step 1: Clone the Repository
git clone https://github.com/<uniquedev200>/OOP_Lab.git

Step 2: Navigate to the Project Folder
cd OOP_Lab

Step 3: Compile and Run the Programs
javac FileName.java
java FileName


Replace FileName with the name of the Java file you want to test.
